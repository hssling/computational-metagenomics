# KDP Publishing Dossier

## Proposed Publishing Path

Primary initial publishing route: **Amazon Kindle Direct Publishing (KDP)** for paperback plus eBook.

## Why This Route

- fastest path from finished manuscript to public availability
- supports both print and digital formats
- suitable for technical handbooks and specialist academic-adjacent books
- allows rapid revision after publication if the manuscript evolves

## Proposed Print Configuration

- trim size: `6" x 9"`
- print style: black ink on white paper
- reading direction: left to right
- interior submission format: DOCX for non-bleed workflow, or PDF if later converted to full print layout

## Current Official KDP Requirements To Observe

The following points are based on current official KDP help pages:

- interior files without bleed may be uploaded as `PDF`, `DOC`, `DOCX`, `RTF`, `HTML`, or `TXT`
- if the interior uses bleed, the manuscript should be submitted as `PDF`
- cover files can be submitted as `PDF`, and cover dimensions should be calculated with the official KDP cover calculator
- all images should be at least `300 DPI`
- files should stay below `650 MB`
- interior content should be supplied as single pages rather than printer spreads
- page numbering must be sequential and correctly sided
- for `6" x 9"` trim, KDP currently lists a page-count range of `24 to 828` for black ink on white paper
- KDP currently lists minimum gutter margins that increase with page count; for `24 to 150` pages, the minimum inside margin is `0.375"` and for `151 to 300` pages it is `0.5"`

## Current Official Workflow Notes

- upload the paperback manuscript from the KDP Bookshelf under paperback content
- launch the KDP Print Previewer before publication to catch file and formatting issues
- order a physical proof copy before final publication when producing a professional print edition
